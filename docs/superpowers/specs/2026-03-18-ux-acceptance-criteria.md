# TradingChest UX 验收标准

基于 TradingView 图表 UX 对比分析制定。所有项目必须通过才算达标。

## A. 指标面板 UX（6 项）

- [x] A1: 搜索框实时过滤（onInput，不是 onChange）
- [x] A2: 分类 Tab 可切换过滤（趋势/波动率/成交量/动量/其他）
- [ ] A3: 分类 Tab 有视觉区分（active 状态高亮、hover 效果）— 需要 CSS
- [ ] A4: 搜索框有搜索图标、清除按钮
- [ ] A5: 指标列表中已选指标有明显的 ✓ 勾选标记（不只是空/实心方块）
- [ ] A6: 面板打开时聚焦搜索框

## B. 绘图工具 UX（5 项）

- [x] B1: 文字标注/便签/标注气泡 — 放置后弹出文字输入
- [x] B2: 右键编辑已放置的文字标注
- [ ] B3: 绘图工具图标有 tooltip 提示工具名称（hover 显示）
- [ ] B4: 选中绘图工具后光标变为十字（crosshair cursor）
- [ ] B5: 删除绘图时有视觉反馈（非静默删除）

## C. 设置面板 UX（4 项）

- [ ] C1: 蜡烛颜色自定义（涨/跌颜色）
- [ ] C2: 背景颜色自定义
- [ ] C3: 十字光标样式选择（十字/仅水平/仅垂直/无）
- [ ] C4: 设置项分组（蜡烛设置/坐标轴/网格/十字光标）

## D. 整体视觉 UX（5 项）

- [ ] D1: 指标面板模态框 CSS 样式 — 搜索框、Tab、列表项样式与 klinecharts-pro 整体风格一致
- [ ] D2: 左侧绘图工具栏图标 hover 时有背景高亮
- [ ] D3: 当前选中的绘图工具有 active 状态指示
- [ ] D4: 指标 tooltip 面板（图例区域）的显隐/设置/关闭图标正确显示
- [ ] D5: 暗色主题下所有新增 UI 元素颜色协调

## E. 功能完整性（4 项）

- [x] E1: 新增指标选中后正确创建子面板并显示数据
- [x] E2: 新增绘图工具能正常绘制
- [ ] E3: 分类 Tab 切换后列表正确过滤
- [ ] E4: 指标参数设置面板可以修改自定义指标参数

## 验收方式

每个项目通过 Playwright 实际操作验证 + 截图确认。不可接受的情况：
1. 功能存在但 UI 不可见/不可操作
2. 样式与 klinecharts-pro 原有风格不协调
3. 交互无视觉反馈
